## Dependency Rules

```
????????????????????????????????????????????????????????????????
?                    CLEAN ARCHITECTURE RULES                   ?
????????????????????????????????????????????????????????????????
?                                                              ?
?  1. Domain Layer depends on: NOTHING                         ?
?     ? Pure business logic                                    ?
?     ? Framework-independent                                 ?
?                                                              ?
?  2. Application Layer depends on: Domain, Shared            ?
?     ? Application business logic                            ?
?     ? Use cases and orchestration                           ?
?                                                              ?
?  3. Infrastructure Layer depends on: Application, Domain,   ?
?     Shared                                                  ?
?     ? External service implementations                      ?
?     ? Framework implementations                             ?
?                                                              ?
?  4. Persistence Layer depends on: Application, Domain,      ?
?     Shared                                                  ?
?     ? Database context                                      ?
?     ? Repository implementations                            ?
?                                                              ?
?  5. API Layer depends on: Application, Infrastructure,      ?
?     Persistence, Shared                                    ?
?     ? HTTP request handling                                 ?
?     ? DI setup                                              ?
?                                                              ?
?  RULE: All dependencies point INWARD ?                      ?
?        (Dependencies flow toward the core)                   ?
?                                                              ?
????????????????????????????????????????????????????????????????
```
